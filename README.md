## Build Setup

- Clone repository

  ```sh
  git clone https://github.com/rafJagCode/portfolio.git
  ```

- Install dependencies

  ```sh
  cd portfolio
  npm install
  ```

- Serve at localhost:3000

  ```sh
  npm run dev
  ```

- Build for production and launch server

  ```sh
  npm run build
  ```

  ```sh
  npm run start
  ```

- Generate static project
 
  ```sh
  npm run build
  ```

  ```sh
  npm run generate
  ```
